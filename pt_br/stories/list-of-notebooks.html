Obter dados

  • how to get full world cup 2014 play-by-play data (en): Bons dados sobre futebol são difíceis de arranjar. Contagens estatísticas básicas estão disponíveis, mas dados completos sobre jogadas (isto é, uma jogada decomposta nos seus componentes individuais: intercepções, cortes, corridas, passes, remates, etc.) são muito raros. E é essa a unidade mais importante num desporto colectivo como o futebol. Por isso imaginem a minha surpresa quando descobri um fantástico conjunto de dados com detalhe completo de cada jogada sobre todos os jogos do Campeonato do Mundo.

  • an exploratory analysis of the world cup final (en): Este notebook mostra como podemos usar dados detalhados de jogadas para analisar um jogo de futebol, utilizando medidas e visualizações costumizadas para melhor entender o desporto.
